# SpaDenRIR
A  simulated dataset of 100 rooms x 10,648 microphones x 5 sources for sound-field reconstruction development and testing

# At the time of writing, the full dataset is still generating.  

## This repo is therefore a small teaser on the data structure. It contains:
- 2 example rooms, generating using the same configurations as described in the article, but limited to a image source order 5.
- A small jupyter notebook showing a few examples on how to handle the dataset.
